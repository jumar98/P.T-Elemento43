# App to Movies Rating

App Web to rating movies of differents cinema.

## Requirements
- PHP >= 7.1 
- MySQL >= 5.7
- Laravel 5.7
### Installation

Install the dependencies and devDependencies and start the server.

```sh
$ cd movie_rating
$ composer install 
$ php artisan serve
```
